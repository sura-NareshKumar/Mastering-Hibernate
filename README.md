# Mastering-Hibernate
Features
--CRUD Operations: Create, Read, Update, and Delete operations implemented with Hibernate.
--HQL Queries: Use of Hibernate Query Language for database operations.
--SQL Queries: Native SQL queries integrated within the Hibernate framework.
--Named Queries: Predefined queries for reusable data access logic.
--Embeddable Objects: Demonstration of embeddable classes to represent value types.
--Annotations: Usage of various Hibernate annotations for mapping Java classes to database tables.
